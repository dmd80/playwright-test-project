Hello. This project is a means to refresh old skills, but also familiarize myself with new ones such as Playwright and AI.

The website used for testing is: https://www.saucedemo.com/.

To run tests, simply clone the repo to your local machine and call upon the test file as shown here: https://playwright.dev/docs/running-tests

After running tests you can check the results by running the following command: `npx playwright show-report`. Results should appear from a localhost url.
